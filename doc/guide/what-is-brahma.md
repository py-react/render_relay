# What is Brahma?

Brahma is a meta-framework designed to bridge the gap between Python's robust backend capabilities and React's dynamic frontend ecosystem.

## Why Brahma?

Traditionally, full-stack React applications rely on Node.js for server-side operations. While effective, this can be limiting for developers who prefer Python's extensive library support, scientific computing tools, and straightforward syntax.

Brahma offers a unique approach:
- **FastAPI Powered**: Leverage the performance and type-safety of FastAPI for your backend.
- **React/Vite Frontend**: Use modern React patterns and Vite's lightning-fast build tool.
- **Unified DX**: A seamless bridge between Python and JS, allowing you to pass data from Python functions directly to React components.
- **File-System Routing**: Familiar routing patterns for both APIs and UI pages.

## Core Philosophy

Brahma believes that you shouldn't have to choose between a powerful Python backend and a modern React frontend. By orchestrating a Node.js bridge for SSR, Brahma gives you the best of both worlds.
